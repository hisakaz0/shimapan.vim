shimapan.vim
==============================================================================

'alternatively coloring' is not cross other vim-highlight and vim-signs
settings. So the plugin works as `filetype plugin`. When you set filetype
`shimapan`, alternatively coloring is activated.

## How to work

- enable: `:ShimapanGo`
- disable: `:ShimapanBye`


## Semilar plugins

1. vim-zebra
2. DynamicSigns
